# Fake News Detection Using Machine Learning

## 📌 Project Overview

This project is a web-based Fake News Detection system developed using Machine Learning and Natural Language Processing (NLP) techniques. The system analyzes news content and predicts whether the news is **Real** or **Fake**.

The application helps users identify misleading information by processing news text and classifying it using trained machine learning models.

---

## 🚀 Features

* Detects fake and real news articles
* User-friendly web interface
* Data preprocessing and text cleaning
* NLP-based feature extraction
* Machine Learning model prediction
* Real-time news classification

## ⚙️ Installation Steps

1. Clone repository

```bash
git clone https://github.com/dhanjisharma-0002/Fake-News-Detection.git
```

2. Move into project directory

```bash
cd Fake-News-Detection
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open browser

```text
http://127.0.0.1:5000/
```
